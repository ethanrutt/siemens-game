Detailed Description

This handles the enter and exit script for the laboratory room which allows players to see what area of the room they are in.

on entering a new area, some text at the bottom will be displayed that tells which section you're in. Room ID Dictionary:

  • 0 -> Automation Station (Top left)
  • 1 -> Synthesis Chamber (Top right)
  • 2 -> Main Hallway (Middle)
  • 3 -> Data Center (Bottom left)
  • 4 -> Digital Command Room (Middle bottom)
  • 5 -> Integration Zone (Bottom right) TOWN SQUARE
  • 10 -> The Reclamation Zone
  • 11 -> Casino
  • 12 -> Scavenger Shop
  • 13 -> Tipsy Tower
  • 14 -> Laboratory
