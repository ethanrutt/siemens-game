Detailed Description

Essentially, want to have a tutorial manager that allows for Sensei to give the player instructions through dialogue. This will be done through a dialogue manager that will be called by the tutorial manager. The tutorial manager will be responsible for managing the tutorial, panning the camera, and other tutorial-related tasks.
