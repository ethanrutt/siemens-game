shuffle() is a template function that shuffles an array This is to be used in the spawnObjects() function so that we can get random positions for the colors and exit spawns The Fisher-Yates algorithm is utilized in this function

Parameters
sourceArrayThe source array that wants to be shuffled
numElementsThe number of elements in the source array

void WireGenerator.Update ( )

Update() is a Unity function that runs on every frame update. This function checks the connections of the wires, and if they are all connected then the gameOverManager member is called to end the level If performance becomes an issue, checkConnection() can be moved to the wire connection behavior function.
