◆ buildSourceMapTree()

function buildSourceMapTree ( input,
loader )

Recursively builds a tree structure out of sourcemap files, with each node being either an OriginalSource "leaf" or a SourceMapTree composed of OriginalSources and SourceMapTrees.

Every sourcemap is composed of a collection of source files and mappings into locations of those source files. When we generate a SourceMapTree for the sourcemap, we attempt to load each source file's own sourcemap. If it does not have an associated sourcemap, it is considered an original, unmodified source file.

◆ remapping()

function remapping ( input,
loader,
options )

Traces through all the mappings in the root sourcemap, through the sources (and their sourcemaps), all the way back to the original source location.

loader will be called every time we encounter a source file. If it returns a sourcemap, we will recurse into that sourcemap to continue the trace. If it returns a falsey value, that source file is treated as an original, unmodified source file.

Pass excludeContent to exclude any self-containing source file content from the output sourcemap.

Pass decodedMappings to receive a SourceMap with decoded (instead of VLQ encoded) mappings.
