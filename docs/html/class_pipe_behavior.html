Detailed Description

This handles the behavior of an individual pipe.

When clicking on a pipe, the pipe rotates 90 degrees. There is also synchronization with the global game state that lives in PipeGenerator. When clicking turn pipes, the sprite image needs to be flipped to keep it consistent with what's displayed at the screen
