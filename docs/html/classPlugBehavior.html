void PlugBehavior.OnTriggerEnter2D ( Collider2D other)
inline

OnTriggerEnter2D() is a Unity function that runs when another Collider2D enters the zone of this gameObject's Collider2D

This function handles the collision of a wireEntry and a wirePlug. If they are the same color then they are locked into place and are marked as connected, otherwise nothing happens.

Parameters
otherThe other Collider2D that is currently colliding with this gameObject
