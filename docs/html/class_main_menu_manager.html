Detailed Description

This handles the login and entering into the introduction cutscene at the start of the game.

This class handles the display of the login modals as well as sending the login api request.

  • On new user creation, the username is checked against a bad word list seen below.
  • Once the player is successfully logged in, we populate PlayerData with the values we get back from the API
  • If the login is unsuccessful, error screens are shown
  • If login is successful, then the player data is populated and the starting cutscene is loaded
