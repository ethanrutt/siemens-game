Direction handles the direction of the pipe in the pipe game. 

This game works by making sure that whatever is flowing through the pipes can only flow in one direction.

For straight pipes, this is rather straight forward, if the pipe is up, then it\textquotesingle{}s in a shape like \texorpdfstring{$\vert$}{|} and it flows upwards.

Turn pipes are a little different. A diagram is drawn below. The \texorpdfstring{$\vert$}{|} and -- indicate the pipe itself, while arrows such as -\/\texorpdfstring{$>$}{>}, \texorpdfstring{$<$}{<}-\/, \texorpdfstring{$^\wedge$}{\string^} \texorpdfstring{$\vert$}{|} \texorpdfstring{$\vert$}{|}, and v, indicate flow.
