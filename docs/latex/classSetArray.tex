\doxysubsection{Detailed Description}
\doxylink{classSetArray}{Set\+Array} acts like a {\ttfamily Set} (allowing only one occurrence of a string {\ttfamily key}), but provides the index of the {\ttfamily key} in the backing array.

This is designed to allow synchronizing a second array with the contents of the backing array, like how in a sourcemap {\ttfamily sources\+Content\mbox{[}i\mbox{]}} is the source content associated with {\ttfamily source\mbox{[}i\mbox{]}}, and there are never duplicates.
