\doxysubsection{Detailed Description}
This class handles character movement. This script will be attached to the player object. 

There are lots of moving parts, since we have separate entities for the player\textquotesingle{}s head, chest, arms, legs, as well as handling movement for all of their cosmetics. Emotes are also handled in this class.
