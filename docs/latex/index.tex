 \hypertarget{index_autotoc_md7}{}\doxysection{\texorpdfstring{Background}{Background}}\label{index_autotoc_md7}

\begin{DoxyItemize}
\item Siemens came to Texas A\&M requesting a project to improve employee morale and productivity in their factory line.
\item Siemens already had a way to track which tasks needed to be done for an order. They had a system in place to have these tasks in some type of queue, load them in to the computers, assign them to a worker, and have the worker track when the tasks are complete.
\item Siemens proposed to link this data to a game, where we could convert these task completion data to in game currency. The employees would also have a way to hang out with each other, and participate in minigames as well as see the leaderboards for who earned the most coins, who has the fastest time in the minigame, etc.
\end{DoxyItemize}

\item They also wanted to be able to sync in-\/game currency with what workers complete in the factory.
