\doxysubsection{Detailed Description}
This handles the interactions with the shop in the town square. 

This will handle initializing the shop, buying items, and keeping track of which items are already owned. These values will be stored in \doxylink{class_player_data}{Player\+Data}, which this class grabs the singleton instance of
