Recursively builds a tree structure out of sourcemap files, with each node being either an {\ttfamily Original\+Source} "{}leaf"{} or a {\ttfamily Source\+Map\+Tree} composed of {\ttfamily Original\+Source}s and {\ttfamily Source\+Map\+Tree}s.

Every sourcemap is composed of a collection of source files and mappings into locations of those source files. When we generate a {\ttfamily Source\+Map\+Tree} for the sourcemap, we attempt to load each source file\textquotesingle{}s own sourcemap. If it does not have an associated sourcemap, it is considered an original, unmodified source file. \Hypertarget{remapping_8mjs_aaf0b70af20c2e866eea5e0830f789096}\index{remapping.mjs@{remapping.mjs}!MapSource@{MapSource}}

{\footnotesize\ttfamily \label{remapping_8mjs_a8cee7ee2719b751cd03554293e4b0258} 
function remapping (\begin{DoxyParamCaption}\item[{}]{input}{, }\item[{}]{loader}{, }\item[{}]{options}{}\end{DoxyParamCaption})}

Traces through all the mappings in the root sourcemap, through the sources (and their sourcemaps), all the way back to the original source location.

{\ttfamily loader} will be called every time we encounter a source file. If it returns a sourcemap, we will recurse into that sourcemap to continue the trace. If it returns a falsey value, that source file is treated as an original, unmodified source file.

Pass {\ttfamily exclude\+Content} to exclude any self-\/containing source file content from the output sourcemap.

Pass {\ttfamily decoded\+Mappings} to receive a \doxylink{classSourceMap}{Source\+Map} with decoded (instead of VLQ encoded) mappings. \Hypertarget{remapping_8mjs_a9181e8bb689170aee97de93276fcb22e}\index{remapping.mjs@{remapping.mjs}!SegmentObject@{SegmentObject}}
