\doxysubsection{Detailed Description}
This class handles the interaction in order to enter the card game. 

This class is attached to the computer object in the card game room. When the player walks into the collider for the computer, the player\textquotesingle{}s interactable is set to "{}cardgame"{} and the interact button is shown. When the player walks away from the collider for the computer, the player\textquotesingle{}s interactable is set to nothing and the interact button is hidden.
