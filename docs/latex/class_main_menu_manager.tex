\doxysubsection{Detailed Description}
This handles the login and entering into the introduction cutscene at the start of the game. 

This class handles the display of the login modals as well as sending the login api request.
\begin{DoxyItemize}
\item On new user creation, the username is checked against a bad word list seen below.
\item Once the player is successfully logged in, we populate \doxylink{class_player_data}{Player\+Data} with the values we get back from the API
\item If the login is unsuccessful, error screens are shown
\item If login is successful, then the player data is populated and the starting cutscene is loaded
\end{DoxyItemize}
