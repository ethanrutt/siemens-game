\doxysubsection{Detailed Description}
This is a wrapper around the Unity\+Web\+Request to make sending web requests easier. 

A Mono\+Behavior child is needed to be passed in to this class, since Web\+Requests in Unity must be Coroutines. There is also a callback for success and failure callback. This is done since the return type of these functions must be {\ttfamily IEnumerator} type, making it a Coroutine, so we can\textquotesingle{}t collect the response from the API call how we normally would.
