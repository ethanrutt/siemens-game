\doxysubsection{Detailed Description}
Manages the tutorial aspect of the inventory, basically if you press the inventory button the inventory panel should open, etc. and once the back button is pressed, it will remove the whole inventory interface and change its main boolean (public) to false, so the tutorial manager script can move on and finish the tutorial.
